Bootstrap-alerts

A jQuery plugin for displaying Bootstrap alerts

View on GitHub Download .zip Download .tar.gz

Basic usage

$('#alert-container').bootstrapAlert({
	type: 'success',
	message: 'This is a success message.',
});

Examples

Success alert Info alert Warning alert Danger alert Info alert with title Info alert without clear

Examples code

<a class="btn btn-success" id="success-alert">Success alert</a>
<a class="btn btn-info" id="info-alert">Info alert</a>
<a class="btn btn-warning" id="warning-alert">Warning alert</a>
<a class="btn btn-danger" id="danger-alert">Danger alert</a>					

<script>
	$('#success-alert').click(function () {
		$('#alert-container').bootstrapAlert({
			type: 'success',
			message: 'This is a <b>success</b> message.'
		});
	});
	$('#info-alert').click(function () {
		$('#alert-container').bootstrapAlert({
			type: 'info',
			message: 'This is a <b>info</b> message.'
		});
	});
	$('#warning-alert').click(function () {
		$('#alert-container').bootstrapAlert({
			type: 'warning',
			message: 'This is a <b>warning</b> message.'
		});
	});
	$('#danger-alert').click(function () {
		$('#alert-container').bootstrapAlert({
			type: 'danger',
			message: 'This is a <b>danger</b> message.'
		});
	});
	$('#info-alert-title').click(function () {
		$('#alert-container').bootstrapAlert({
			type: 'info',
			title: 'My title',
			message: 'This is a <b>info</b> alert with title.'
		});
	});
	$('#info-alert-no-clear').click(function () {
		$('#alert-container').bootstrapAlert({
			type: 'info',
			clear: false,
			message: 'This is a <b>info</b> alert without clear container.'
		});
	});
</script>

Options

Property Type Description Default Values
type string Required. Type of alert empty 'success', 'info', 'warning' or 'danger'
dismissible bool Shows the dismiss button in the alert box. true
title string Shows the title alert box. empty
message string Required. Message of the alert empty
clear bool Clears the container before shows the alert true
